Projeto Físico HidroSafe

Visão Geral do Sistema

O sistema físico HidroSafe é composto por dois módulos principais que trabalham em conjunto para monitorar, analisar e responder a riscos de enchentes em tempo real. Nossa solução combina sensores de alta precisão, processamento local e atuadores físicos para criar um sistema completo de prevenção de enchentes.

Diagrama esquemático do sistema HidroSafe

Módulo de Monitoramento Local

Este módulo é fixado em locais estratégicos próximos a rios, canais ou bueiros. Ele é responsável pela coleta contínua de dados ambientais e pela análise em tempo real das condições que podem levar a enchentes.

Componentes:

  • Microcontrolador Arduino Uno - Processa os dados coletados e executa a lógica de análise de risco.
  • Sensor ultrassônico HC-SR04 - Posicionado acima da superfície da água, mede constantemente o nível de rios ou canais com precisão de ±3mm.
  • Sensor DHT22 - Coleta continuamente os dados de umidade e temperatura do ar. Quando a umidade está próxima ou igual a 100%, o sistema reconhece o início de uma chuva significativa.
  • Real Time Clock RTC DS3231 - Mantém registro preciso de data e hora para os dados coletados.
  • Display LCD (I2C) - Exibe informações em tempo real sobre o nível da água e condições ambientais.
  • Sistema de alerta local - LEDs e buzzer para alertas visuais e sonoros.
  • Teclado matricial - Permite configuração e interação local com o sistema.
  • Sistema de energia - Bateria recarregável e painel solar para funcionamento autônomo.

Funcionamento:

Durante a chuva, os dados são analisados de forma contínua, detectando elevação gradual ou repentina do nível de água. Se o nível atingir valores de risco definidos (ex: 1,8m), o sistema emite alertas visuais e sonoros em tempo real. Quando o risco se mantém por um tempo crítico (ex: mais de 15 minutos de elevação constante), o sistema aciona diretamente o módulo de controle hidráulico.

Módulo de Controle Hidráulico (Atuador)

Este módulo é responsável por atuar sobre barreiras físicas como pequenas comportas ou portões hidráulicos, que regulam o fluxo de água e ajudam a prevenir o transbordamento. Este módulo é apresentado como um conceito demonstrativo do potencial do sistema.

Componentes:

  • Servo motor MG996R - Responsável pelo movimento físico das comportas, com torque suficiente para operar pequenas barreiras.
  • Sensores de posição - Monitoram o estado atual das comportas (aberta/fechada).
  • Sistema de energia - Bateria recarregável e painel solar para funcionamento autônomo.

Funcionamento:

Com base nos dados analisados pelo módulo de monitoramento, o servo motor é acionado para abrir ou fechar as comportas de acordo com a necessidade. O sistema mantém registro do estado das comportas para garantir operação adequada em diferentes condições de nível de água.

Modelo Matemático de Funcionamento

O sistema HidroSafe utiliza um modelo matemático baseado em polinômios de grau 3 para analisar e prever o comportamento do nível de água durante eventos de chuva. Este modelo permite:

  • Calcular a taxa de elevação do nível de água em tempo real
  • Prever o tempo até que o nível atinja a cota de transbordamento
  • Determinar o momento ideal para acionamento das comportas

A lógica de acionamento das comportas considera três critérios principais:

  1. Comparação do nível atual com a cota de transbordamento
  2. Análise da taxa de elevação (derivada da função)
  3. Previsão do tempo até ultrapassagem da cota

Instalação e Manutenção

O sistema HidroSafe foi projetado para ser de fácil instalação e baixa manutenção:

  • Instalação em postes ou estruturas existentes próximas aos corpos d'água
  • Calibração inicial simples com assistência técnica
  • Manutenção preventiva anual recomendada
  • Vida útil estimada de 5 anos para os componentes eletrônicos
  • Substituição simples de módulos em caso de falha
